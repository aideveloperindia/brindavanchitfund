# Brindavan Chits Website

A modern, responsive website for Brindavan Chits Karimnagar (India) Pvt. Ltd., a trusted chit fund institution established in 2007.

## Features

- **Modern Design**: Clean, professional layout with green color scheme
- **Responsive**: Mobile-first design that works on all devices
- **Fast Performance**: Built with Next.js 14 for optimal speed
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Accessible**: WCAG compliant design for all users

## Sections

1. **Home**: Company introduction with key statistics
2. **About Us**: Company history, board of directors, and mission
3. **Chit Schedules**: Interactive table of current chit schemes
4. **Team**: Professional team members display
5. **How It Works**: Step-by-step process explanation
6. **Contact**: Branch locations and contact information
7. **Quotation**: Detailed project information page

## Technologies Used

- **Next.js 14**: React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **Responsive Design**: Mobile-first approach

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser
